You'll never post so fast... // More details on the documentation page // This extension works with WP 3.4+

Blogs Settings

Enter your wordpress settings to connect and post on your blog

- Wordpress Blog Url : The home url of the wordpress blog where to send the generated post
- Example : http://yourblog.com
- Wordpress User name : the user that will be used to post from Postek
- Wordpress User pass : the pass of the user

Crawl Settings
- Minimum Image width : The minimum width of pictures automatically fetched in the page
- Enter an integer (ex: 350)
- If empty, 500px will be used by default
- Minimum Image height : The minimum height of pictures automatically fetched in the page
- Enter an integer (ex: 350)
- If empty, 200px will be used by default

Posts Settings
- Image/Video width : The width that will be used for images / videos in the generated post
- Enter an integer (ex: 580)
- If empty, 560px will be used by default
- For videos, height is calculated based on a 16:9 ratio
- For images, height is calculated based on the picture ratio
Save your settings
